# keyboard-typing-visualizer
A simple JavaScript project that displays user keystrokes in real time using DOM manipulation and event listeners.

## 🚀 Features
- Captures keyboard input using event listeners
- Updates UI instantly using DOM manipulation
- Clean and responsive design

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript (DOM + Events)

## 📚 What I Learned
- How to use keydown event
- How to manipulate DOM dynamically
- Handling user input in real-time

## 🔮 Future Improvements
- Add animations
- Show key codes
- Support key combinations (Ctrl, Shift, etc.)
